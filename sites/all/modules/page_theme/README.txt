Module
======
Page Theme


Description
===========
This module allows to use different themes than the site default on specific
pages.


Requirements
============
Drupal 7.x


Installation
============
1. Move this folder into your modules directory.
2. Enable it from Administer >> Modules >> Theme related.


Configuration
=============
Configure it at Administer >> Structure >> Page theme.


Projekt page
============
http://drupal.org/project/page_theme
